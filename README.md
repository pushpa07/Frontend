# FRONTEND PROJECT
In this project we are mainly concentrated in frontend things.It is a college prototype project which mainly contains home page,registration page,about us page,login page,css page.In home page we are giving the all elements which are contained in home page of a college like logo of a college,images of college and some links which are connected in all pages.In the css page it contains the style of the container boxes color of the page and so on.In the registration page it contains all the empty boxes where we have to register all our details to get admission to our college.
